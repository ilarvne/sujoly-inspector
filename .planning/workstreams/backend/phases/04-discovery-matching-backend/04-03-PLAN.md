    **A. Create STAC service (apps/api/src/api/services/stac_service.py):**

    Lightweight STAC-like catalog stored in MinIO + metadata in a simple dict structure.
    For hackathon MVP, we store STAC items as JSON in MinIO and serve them via API.
    No separate STAC server needed — TiTiler reads COGs directly from MinIO.

    ```python
    class StacService:
        async def create_collection(self, collection_id: str, description: str, 
                                     bbox: list[float], temporal: list[str]) -> dict:
            """Create a STAC collection in MinIO.
            Stored as JSON at stac/collections/{collection_id}/collection.json
            """
            
        async def add_item(self, collection_id: str, item: dict) -> dict:
            """Add a STAC item to a collection.
            Item must follow STAC Item spec minimally:
            - id, geometry, bbox, properties.datetime
            - assets with COG hrefs pointing to MinIO
            Stored at stac/collections/{collection_id}/items/{item_id}.json
            """
            
        async def search_items(self, collection_id: str = None, 
                                bbox: list[float] = None,
                                datetime_range: str = None) -> list[dict]:
            """Search STAC items by collection, bbox, datetime.
            For MVP: list items from MinIO and filter in-memory.
            """
            
        async def get_titiler_url(self, cog_path: str) -> str:
            """Generate TiTiler URL for a COG asset.
            Returns URL like: http://titiler:8081/cog/tiles/{z}/{x}/{y}.png?url={cog_path}
            """
    ```

    **A. Create OCR service (apps/api/src/api/services/ocr_service.py):**

    For hackathon MVP: a stub OCR service that extracts text from uploaded documents.
    Uses a simple pattern-matching approach for Russian/Kazakh hydraulic structure passports.
    Can be upgraded to Tesseract/EasyOCR/PaddleOCR for production.

    ```python
    class OcrService:
        async def extract_text(self, file_bytes: bytes, filename: str) -> dict:
            """Extract text from uploaded document.
            
            For MVP: if the file is a text-based format, read directly.
            For images/PDFs: return placeholder with confidence=LOW,
            noting that full OCR requires Tesseract/EasyOCR integration.
            
            Returns: {
                "text": extracted text,
                "confidence": "HIGH"|"MEDIUM"|"LOW",
                "language": "ru"|"kk"|"unknown",
                "entities": list of extracted entities (names, numbers, dates)
            }
            """
            
        async def extract_entities(self, text: str) -> list[dict]:
            """Extract named entities from Russian/Kazakh text.
            
            Pattern-match for:
            - Structure names (after "Наименование:", "Атауы:")
            - Years (4-digit numbers after "год", "жыл")
            - District names (after "Район:", "Аудан:")
            - Condition ratings (after "Состояние:", "Жағдай:")
            
            Returns list of {"type": str, "value": str, "confidence": str}
            """
            
        async def process_document(self, document_id: uuid.UUID) -> dict:
            """Process an existing document through OCR.
            
            1. Load document from DB
            2. Fetch file from MinIO
            3. Run extract_text
            4. Store results in document metadata
            5. If entities found, create candidate from OCR data
            """
    ```
