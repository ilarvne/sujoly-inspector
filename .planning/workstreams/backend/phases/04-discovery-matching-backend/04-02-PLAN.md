    **A. Create matching_service.py:**

    ```python
    class MatchingService:
        async def match_candidate(self, candidate: CandidateModel) -> CandidateMatchResult:
            """Match a single candidate against the registry.
            
            Hierarchical matching (DISC-02):
            1. Spatial proximity: ST_DWithin(candidate.geometry, structure.geometry, 500m)
               → Find structures within 500m radius
            2. Name similarity: pg_trgm similarity(candidate.name, structure.name_ru/kk/en)
               → similarity() > 0.3 threshold
            3. Attribute comparison: type, district, water_source match
            4. Confidence scoring: weighted combination of all signals
            
            Match state assignment (DISC-03):
            - matched: spatial < 100m AND name_sim > 0.7 → HIGH confidence
            - likely_match: spatial < 500m AND name_sim > 0.3 → MEDIUM confidence  
            - new_candidate: no structures within 500m → LOW confidence
            - conflict: spatial < 100m but name/type mismatch → LOW confidence
            """
            
        async def match_all_unmatched(self) -> list[CandidateMatchResult]:
            """Run matching for all unmatched candidates.
            
            1. Load all candidates with match_status="unmatched"
            2. For each, run match_candidate
            3. Update candidate records with results
            4. Return results
            """
            
        async def compute_confidence(self, spatial_dist: float, name_sim: float, 
                                      attr_matches: int) -> tuple[str, float]:
            """Compute confidence level and score from signals.
            
            Weighted formula:
            score = 0.4 * name_sim + 0.3 * (1 - spatial_dist/500) + 0.3 * (attr_matches/3)
            
            Level:
            HIGH if score >= 0.7
            MEDIUM if score >= 0.4
            LOW otherwise
            """
    ```

    The spatial query uses PostGIS ST_DWithin via SQLAlchemy:
    ```python
    from geoalchemy2 import functions as geo_func
    stmt = select(StructureModel).where(
        geo_func.ST_DWithin(
            StructureModel.geometry,
            candidate.geometry,
            500  # meters
        )
    )
    ```

    Name similarity uses pg_trgm via raw SQL or func.similarity():
    ```python
    from sqlalchemy import func, text
    # similarity() requires pg_trgm extension (added in migration 0008)
    stmt = select(
        StructureModel,
        func.similarity(StructureModel.name_ru, candidate.name).label("name_sim")
    )
    ```
