    **A. Create CandidateModel (apps/api/src/api/models/candidate.py):**

    ```python
    class CandidateModel(Base):
        __tablename__ = "candidates"

        id: UUID primary key
        name: String(500) — candidate name from source
        source_type: String(50) — "osm" | "satellite" | "ocr" | "manual"
        source_id: String(255) — OSM node/way ID, or external reference
        geometry: Geometry(Point, srid=4326) — candidate location
        match_status: String(50) — "unmatched" | "matched" | "likely_match" | "new_candidate" | "conflict" | "rejected"
        matched_structure_id: UUID FK to structures.id (nullable) — set when matched/linked
        confidence: String(10) — "HIGH" | "MEDIUM" | "LOW"
        confidence_score: Float — numeric 0.0-1.0
        evidence: JSONB — dict of evidence sources and their contributions
          e.g. {"osm": {"tags": {...}, "distance_m": 50}, "name_similarity": 0.85}
        district: String(255) nullable
        water_source: String(255) nullable
        type: String(100) nullable — inferred structure type
        review_notes: Text nullable
        reviewed_by: UUID FK to users.id nullable
        reviewed_at: DateTime(timezone=True) nullable
        provenance_id: UUID FK to provenance.id
        created_at, updated_at: DateTime(timezone=True)

        __table_args__ = (
            CheckConstraint("source_type IN ('osm', 'satellite', 'ocr', 'manual')", name="ck_candidate_source_type"),
            CheckConstraint("match_status IN ('unmatched', 'matched', 'likely_match', 'new_candidate', 'conflict', 'rejected')", name="ck_candidate_match_status"),
            CheckConstraint("confidence IN ('HIGH', 'MEDIUM', 'LOW')", name="ck_candidate_confidence"),
            Index("ix_candidates_match_status", "match_status"),
            Index("ix_candidates_source_type", "source_type"),
        )
    ```
