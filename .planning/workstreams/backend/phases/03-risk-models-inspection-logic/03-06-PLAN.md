## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → API export endpoints | JWT required (viewer+ role). Query params (filters, lang) are user input |
| API → MinIO | Photo download for PDF base64 embedding — MinIO SDK handles auth |
| WeasyPrint → system | HTML→PDF rendering via Pango system libraries — no network access |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-18 | Tampering | CSV export — formula injection | mitigate | Prefix cells starting with =, +, -, @ with single quote (') before writing to CSV. Standard CSV injection mitigation. Implement in export_structures_csv: check first char of each cell value, prepend ' if it matches [=+\-@]. |
| T-03-19 | Tampering | Export query params — bbox injection | mitigate | Reuse existing _apply_bbox_filter from structure_service.py which validates 4 numeric values and uses func.ST_MakeEnvelope. ValueError → HTTP 400. |
| T-03-20 | Denial of Service | PDF generation blocks event loop | mitigate | Use asyncio.to_thread() for WeasyPrint HTML.write_pdf() call. Prevents blocking the async event loop during PDF generation (Pitfall 5 from RESEARCH.md). |
| T-03-21 | Information Disclosure | PDF contains MinIO photo data | accept | Photos are inspection evidence intended for the report. Presigned URL access is already gated by RBAC. PDF download requires viewer+ role. No additional disclosure risk. |
</threat_model>
