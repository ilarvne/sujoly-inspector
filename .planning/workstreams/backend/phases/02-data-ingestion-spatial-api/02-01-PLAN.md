## Phase Goal

**As a** catalog operator, **I want to** ingest the Kazvodhoz canal registry into PostGIS and access structures via REST and OGC APIs with multilingual search, **so that** I can query and manage hydraulic structures from both the application and external GIS tools.

## Plan 01 Slice: Data Ingestion

**As a** catalog operator, **I want to** trigger ingestion of the Kazvodhoz spreadsheet and have canal records loaded into PostGIS, **so that** the catalog has a baseline dataset to search and display.
