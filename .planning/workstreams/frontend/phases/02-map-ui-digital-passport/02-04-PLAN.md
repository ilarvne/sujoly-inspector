## Phase Goal

**As a** water resource manager, **I want to** view all hydraulic structures on an interactive map with their digital passports and portfolio analytics, **so that** I can assess the condition and inspection status of infrastructure across Zhambyl Oblast.

---

<objective>
Replace the filter panel stub from Plan 01 with a full filter panel. The panel provides 5 dropdown selectors (district, basin, structure type, condition, inspection status) and a reset button. Filter changes update the Zustand filter store, which triggers TanStack Query to recompute the filtered GeoJSON, which reactively updates both the map markers and the dashboard charts.

Purpose: MAP-05 (filter map and dashboard by district, basin, type, condition, inspection status) gives users the ability to narrow their view to specific subsets of structures.
Output: Working filter panel on the /map page that affects both map and dashboard data in real-time.
</objective>
