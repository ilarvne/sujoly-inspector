## Phase Goal

**As a** water resource manager or inspector, **I want to** see a full chat interface at /copilot with message bubbles, suggested prompts, interactive cards, and clickable source citations, **so that** I can interact naturally with the AI assistant.

---

<objective>
Build all chat UI components and replace the /copilot stub page with the full chat interface. Components include message rendering, input with send, suggested prompts, source citations, and four interactive card types (StructureCard, RiskBreakdownCard, InspectionCard, ReportCard).
</objective>

<description>
Create all copilot chat components, replace the stub /copilot page, verify build passes.
</description>
<instructions>
1. Create components/copilot/ directory with:
   - chat-message.tsx: Renders user (right-aligned, primary bg) vs assistant (left-aligned, card bg) messages. Avatar icons (UserIcon for user, SparklesIcon for assistant). Timestamp. Renders cards inline if present. Renders source citations if present. Streaming indicator (animated dots) when isStreaming.
   - chat-input.tsx: Textarea + send Button. Enter to send (Shift+Enter for newline). Disabled when isStreaming. Placeholder from i18n.
   - source-citation.tsx: Renders array of CopilotSource as clickable Badge components. Color-coded by source type. Clicking sets selectionStore.selectedId to navigate to structure on map (via router push to /map).
   - structure-card.tsx: Compact Card showing structure name, ID, type, condition badge, district. Clickable to open passport.
   - risk-breakdown-card.tsx: Card showing overall risk score with Progress bar, risk level badge, component breakdown mini-bars.
   - inspection-card.tsx: Card showing inspection date, inspector, findings snippet, condition badge.
   - report-card.tsx: Card showing report title, summary, structure count.
   - suggested-prompts.tsx: Row of clickable prompt chips from getSuggestedPrompts(locale). Only shown when messages array is empty.
   - copilot-chat.tsx: Main orchestrator. Uses useChatStore for messages/isStreaming/sendMessage/clearChat. Uses useLocale for language. ScrollArea for message list with auto-scroll-to-bottom. Renders SuggestedPrompts when empty. Renders ChatMessage for each message. Renders ChatInput at bottom. Clear chat button in header.

2. Replace app/[locale]/copilot/page.tsx:
   - Keep as async server component (await params, setRequestLocale, getTranslations)
   - Render page header with title/subtitle from i18n
   - Render <CopilotChat /> client component

3. Run build and verify all pages compile.
